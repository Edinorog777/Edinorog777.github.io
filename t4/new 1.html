Windows XP

Windows XP (кодова назва при розробці — Whistler; внутрішня версія — Windows NT 5.1) — операційна система сімейства Windows NT від компанії Microsoft. Вона була випущена 25 жовтня 2001 року і є розвитком Windows 2000 Professional. Назва XP походить від англ. experience (досвід, враження, від прикметника професійний). Назва увійшла до практики використання, як професійна версія. На відміну від попередньої системи Windows 2000, яка поставлялася як в серверному, так і в клієнтському варіантах, Windows XP є виключно клієнтською системою. Її серверним варіантом є випущена пізніше система Windows Server 2003. Windows XP і Windows Server 2003 побудовані на основі одного і того ж ядра операційної системи, в результаті їхній розвиток і оновлення йде більш-менш паралельноp
